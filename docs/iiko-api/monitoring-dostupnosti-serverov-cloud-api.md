В качестве endpoint'а используется URL **https://api-ru.iiko.services**.

Реализован метод, который интеграторы могут вызывать для проверки доступности серверов Cloud API - **https://api-ru.iiko.services/thisistheway**.

Если kubernetes работает – в ответ вернётся "200 Ок", если kubernetes не работает – вернётся "503 error"
