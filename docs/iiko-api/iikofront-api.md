Функциональность iikoFront можно расширить с помощью плагинов. С помощью специального программного интерфейса API можно изменять поведение приложения в некоторых сценариях:

* Печатать в чеках дополнительную информацию.
* При вводе заказа отображать его состав на мониторе покупателя.
* Добавить интеграцию с внешними системами приема заказов: веб-сайты, электронные меню, мобильные терминалы официантов.
* Собирать статистику и строить отчеты.
* Подключать внешние платежные системы и многое другое.
