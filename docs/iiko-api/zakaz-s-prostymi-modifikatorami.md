Модификаторы бывают простыми и в составе группы, обязательными и необязательными. Если блюдо содержит модификаторы, то блюдо со всеми его модификаторами нужно выгружать в API.

В этой статье рассмотрим простые модификаторы.

Модификаторы в составе блюда отображаются в API следующим образом:

```json
{
"modifiers": [
{
"maxAmount": 3, //Максимальное кол-во. Нельзя добавить в заказ блюдо с большим кол-во модификаторов!
"minAmount": 1, //Минимальное кол-во.Нельзя добавить в заказ блюдо с меньшим кол-вом модификаторов! Если required:true , то как минимум 1.
"modifierId": "01ec7323-7659-4531-9502-9a2b129dbfcc", //Уникальный id модификатора.
"required": true, //Является обязательным! Нельзя добавить блюдо в заказ без обязательного модификатора!
"defaultAmount": 2 //Кол-во по-умолчанию , которое добавляется в заказ с блюдом при заказе на iiko Front
},
{
"maxAmount": 4,
"minAmount": 0,
"modifierId": "aed0e5e1-6e00-4aa9-b50e-ef77d81e10e4",
"required": false,//Является не обязательным.Можно добавить блюдо в заказ без данного модификатора.
"defaultAmount": 0
}
]
```
