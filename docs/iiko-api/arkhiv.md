Архив [iiko.biz API](/articles/api-documentations/iiko-biz-api)

С 1 июля 2022 года мы прекратили предоставлять API iiko.biz для создания новых интеграций с iiko.

Все новые интеграции с внешними системами доставки создаются на базе [iikoCloud API](/articles/api-documentations/iikocloud-api).

Поддержка уже реализованных интеграционных решений в рамках существующих лицензионных контрактов осуществляется до конца 2022 года.
