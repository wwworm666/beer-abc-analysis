Вы получите демонстрационную лицензию NFR, которая содержит все доступные для интеграции API лицензии. Так же вам будет предоставлена инструкция, по который вы сможете развернуть у себя среду для тестирования.

## Получение демо-стенда

После запроса демо-стенда вы получите электронное письмо со следующими данными:

**Примечание 1:** CrmID, домен, версия приведены в качестве примера. У вас будут другие данные. Эти данные будут использоваться в руководстве, но вам нужно будет заменить их данными, которые вы получите по электронной почте.
