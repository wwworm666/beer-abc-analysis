**Внимание! С 1 июля 2022 года мы прекратим предоставлять API iiko.biz для создания новых интеграций с iiko. Все будущие интеграции с внешними системами доставки будут создаваться на базе iiko Cloud API. При этом до конца 2022 года мы будем продолжать поддержку уже реализованных интеграционных решений в рамках существующих лицензионных контрактов.**

## Правильная версия 

Убедитесь в том, что у вас установлена iikoRMS версии 4.2 или выше. Если у вас более старая версия, то обратитесь к дилеру в вашем регионе и запросите обновление.

## Регистрация в iiko.Biz 

Для регистрации в iiko.Biz зайдите в iikoOffice, выберите раздел **Обмен данными → iiko**.**Biz → Состояние подключения**. Система предложит вам выполнить регистрацию на портале iiko.Biz.

![](/resources/Storage/api-documentations/topic-img0001.png)

Введите название предприятия, физический адрес, ФИО администратора, email и пароль для входа в iiko.Biz, нажмите кнопку **Зарегистрировать**.

![](/resources/Storage/api-documentations/topic-img0002.png)

Убедитесь в том, что все проверки выполнены успешно. При необходимости перезапустите их. Для этого нажмите кнопку **Перезапустить**рядом с соответствующей проверкой.
![](/resources/Storage/api-documentations/topic-img0003.png)
## Подключение API-логина 

Чтобы получить логин и пароль для использования API iikoBiz необходимо обратиться в обслуживающую организацию, чтобы они развернули соответствующий тикет в техническую поддержку iiko. Также можно написать на почту api@iiko.ru.

## Подготовка меню для внешних систем 

Откройте раздел **Обмен данными → iiko**.**Biz → Выгрузка меню**. Выберите те товары, которые должны отображаться на сайте, задайте им картинки и желаемые имена для отображения.

После формирования меню нажмите **Выгрузить в API**.

Подробнее о формировании меню для выгрузки см. в статье [Меню для внешних систем](/smart/project-iikooffice/topic-905).
![](/resources/Storage/api-documentations/topic-img0005.png)
## Выгрузка городов 

Перейдите в справочник городов. Если он пустой, то добавьте те города, которые входят в вашу зону доставки. После этого нажмите кнопку **Выгрузить в API**.
![](/resources/Storage/api-documentations/topic-img0006.png)
