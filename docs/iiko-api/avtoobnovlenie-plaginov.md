**Цель**

Обеспечить автоматическое обновление плагинов на кассе.
 
**Принцип работы**

* Для каждой версии фронтового API выпускается соответствующая версия плагина.
* На кассу автоматически устанавливается плагин с максимально поддерживаемой версией API, совместимой с текущей конфигурацией.

**Ограничения и лицензии**

* **Лицензия**: на фронте должна быть активна лицензия с module ID `0021035101`.
 (Лицензия выдается в CRM и на текущий момент включена для всех стран.)
* **Совместимость**: требуется версия iiko **7.6 или выше**.
* **Ограничения:**
    * Невозможно отключить обновление для отдельного ресторана.
    * Нельзя ставить разную версию плагина разным клиентам.

### Архитектура

* База - NuGet,

* Возможность работы с несколькими источниками с пакетами,

* Реализован сценарий автоустановки,

* Автообновление (в т.ч. установленных вручную),

* Переход между версиями iikoFront API,

* Миграция настроек и данных.
