**iikoCloud API** — это сервис, через который компоненты iiko обмениваются информацией друг с другом или с внешними системами. Его можно использовать:

* для интеграции со своим сайтом — заказы с сайта будут передаваться на кассу ресторана;
* для интеграции с внешними сервисами по доставке еды, такими как СберМаркет или Яндекс.Еда;
* для подключения мобильного приложения Driver App.

Текущая версия Cloud API работает только с заказами доставки, начиная с версии iiko 7.9.7.

Настройки Cloud API доступны пользователям лицензий iikoCloud и iikoClassic.

| ![Information](/resources/Storage/api-documentations/project-iikoweb/info.png) | Начиная с версии iiko 7.0, к настройкам Cloud API можно перейти из классического офисного приложения iikoOffice. Для этого в главном меню выберите пункт **Обмен данными** → **Настройка iikoTransport**. После этого в браузере откроется страница настроек Cloud API в веб-офисе iiko. |

## Лицензирование

Лицензия Cloud API, открывающая возможности интеграционного API, включена во все облачные тарифы iikoCloud.

| ![Warning](/resources/Storage/api-documentations/api-documentations/warning.png) | Пользователям локальных тарифов iikoClassic (lifetime тарифы) необходимо дополнительно приобрести такую лицензию (1 лицензия на 1 ресторан) **до 31.12.2025 г.** |
