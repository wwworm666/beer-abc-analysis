Существует три вида iiko API

[API iikoServer](/articles/api-documentations/~iikoserver-api) - "серверный" API, позволяет работать с номенклатурой, отчетами, документами, данными сотрудников, приказам.

[API iikoFront](/articles/api-documentations/~iikofront-api)- данный API позволяет автоматизировать и расширять функциональность кассового интерфейса.

[API iikoCloud](/articles/api-documentations/~iikocloud-api) - API для работы с заказами на стол, доставочными заказами и банкетами.
