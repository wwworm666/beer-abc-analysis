| Название | Содержит | Что можно сделать |
| --- | --- | --- |
| **iikoAPIServer** | API корпорации<br> <br>API OLAP-отчетов<br> <br>API документов<br> <br>API событий<br> <br>API сотрудников | Используя данное апи можно реализовать:<br><ul> <li>Кастомные и аналитические системы отчетов.</li> <li>Электронный документооборот с поставщиками.</li> <li> Системы расчета заработной платы и контроля учета рабочего времени.</li> <li> Интеграции с внешними системами видео наблюдения и мониторинг за опасными операциями.</li> </ul> |
