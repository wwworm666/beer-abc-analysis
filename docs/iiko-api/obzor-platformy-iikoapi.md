Благодаря своей универсальности и гибкости iiko API позволяет разработчикам создавать собственные приложения и решения на основе платформы iiko, что обеспечивает ещё большую адаптивность системы.

Основные сферы применения API iiko включают:

1. Управление заказами — автоматизация приема, обработки и передачи заказов между внешними системами и iiko.
2. Интеграция с системами учета и ERP — обмен данными о продажах, запасах и финансовых операциях для повышения точности учета.
3. Лояльность — автоматизация программ лояльности и аналитики по клиентам.

Существует три вида API:

[API iikoServer](/articles/api-documentations/~iikoserver-api) - "серверный" API, позволяет работать с номенклатурой, отчетами, документами, данными сотрудников, приказами.

[API iikoFront](/articles/api-documentations/~iikofront-api)- данный API позволяет автоматизировать и расширять функциональность кассового интерфейса, разрабатывать интеграции с [платежными системами](/articles/special-iiko/payment-systems), интеграции с [гостиничными системами](/articles/special-iiko/hotels) или [системами лояльности](/articles/special-iiko/loyalty), а также системы оперативного мониторинга.

[API iikoCloud](/articles/api-documentations/~iikocloud-api) - API для работы с заказами на стол, доставочными заказами и банкетами.
