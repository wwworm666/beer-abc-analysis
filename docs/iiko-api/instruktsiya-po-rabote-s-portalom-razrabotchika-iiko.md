Портал разработчика iiko — это единая точка входа для работы с API iiko.

Позволяет:

* обеспечить стабильный доступ к API;
* получить доступ к новым сервисам и возможностям платформы iiko;
* использовать API новых методов, которые доступны только зарегистрированным сервисам;
* Производительность интеграции за счёт настроек со стороны iiko.

## Регистрация
* Для регистрации необходимо перейти по ссылке [https://public-api.iikoweb.ru/portal](https://public-api.iikoweb.ru/portal)

При первом открытии вы увидите главную страницу:
![](/resources/Storage/api-documentations/instruktsiya-po-rabote-s-portalom-razrabotchika-iiko/instruktsiya-po-rabote-s-portalom-razrabotchika-iiko-2026-06-01-12.png)
 
* Нажмите кнопку **«Начать работу»**, чтобы перейти на страницу регистрации.
* Или на кнопку **«Войти»** в верхнем меню сайта и перейдите на блок **«Регистрация»**.

### 1. Заполните форму регистрации.
![](/resources/Storage/api-documentations/instruktsiya-po-rabote-s-portalom-razrabotchika-iiko/instruktsiya-po-rabote-s-portalom-razrabotchika-iiko-2026-06-02-1.png)

* Введите ваш рабочий адрес электронной почты в поле **Email** (это обязательное поле).
* Придумайте пароль длиной не менее 8 символов и укажите его в поле **Пароль** — вы можете нажать на иконку справа, чтобы отобразить вводимые знаки.
* Повторите этот же пароль в поле **Подтвердите пароль** для его подтверждения.
* В завершение обязательно установите флажок согласия с обработкой персональных данных.

После нажатия кнопки «Зарегистрироваться» на вашу почту будет отправлено письмо для верификации аккаунта. При этом в личном кабинете автоматически откроется окно «Настройки компании». Оно будет появляться при каждом входе до тех пор, пока вы не заполните все обязательные поля.
![](/resources/Storage/api-documentations/instruktsiya-po-rabote-s-portalom-razrabotchika-iiko/instruktsiya-po-rabote-s-portalom-razrabotchika-iiko-2026-06-01-4.png)

### 2. Подтверждение email

* Откройте почтовый ящик, который вы указали при регистрации, и найдите письмо от нашего портала.
* Если письма нет во входящих, обязательно проверьте папку «Спам».
* Для завершения регистрации перейдите по ссылке или нажмите кнопку подтверждения внутри письма.

## Возможные проблемы и способы их решения

| Проблема | Возможная причина | Решение |
| --- | --- | --- |
| **«Email уже зарегистрирован»** | Аккаунт с таким адресом уже существует в системе. | Войдите на портал через форму **«Вход»** (Login) или воспользуйтесь функцией восстановления пароля. |
| **Не приходит письмо для верификации** | Письмо попало под фильтры почты или отправка заняла чуть больше времени. | Проверьте папку **«Спам»** или подождите 2–3 минуты. |
| **Кнопка «Добавить» горит серым (неактивна)** | Ваш адрес электронной почты еще не подтвержден. | Перейдите по ссылке из приветственного письма для подтверждения email. |
| **Появился красный текст под полем** | Введенные данные не соответствуют требованиям системы (ошибка валидации). | Проверьте, совпадают ли пароли, заполнены ли обязательные поля и корректен ли формат данных. |
| **Ошибка при отправке заявки** | Произошел кратковременный сбой в сети или на стороне сервера. | Обновите страницу (нажмите **F5**) и повторите попытку. Если ошибка повторится, запишите её текст и обратитесь в поддержку. |

## Глоссарий терминов

| Термин | Значение |
| --- | --- |
| **appId** | Уникальный идентификатор приложения, используемый для выполнения API-вызовов. |
| **Client Secret** | Секретный ключ авторизации. Обладает конфиденциальным статусом — храните его в тайне и не передавайте третьим лицам! |
| **IntegratorsKit** | Специализированная программа поддержки сторонних разработчиков от iiko. |
| **Валидация** | Автоматическая проверка корректности введенных данных. Если данные введены неверно, под полем появится подсказка красного цвета. |
