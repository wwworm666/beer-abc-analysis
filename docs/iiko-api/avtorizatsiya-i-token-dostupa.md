Для работы с API используется временный токен доступа, срок жизни которого 15 минут.

После [подключения API iikoDelivery](/articles/api-documentations/kak-podklyuchit-vneshniy-api) на портале iiko.biz, можно проверить получение токена.

Например, получить токен можно GET-запросом:

https://iiko.biz:9900/api/0/auth/access\_token?user\_id=bus1ness&user\_secret=bus1ness

где:

* user\_id — ваш API-логин.
* user\_secret — ваш пароль для API. Не путайте с логином и паролем для входа на портал iiko.biz.

Для POST-запросов необходимо указывать Content-Type: application/json.

Для стабильной работы сайта или приложения всегда проверяйте успешность запроса. Убедитесь, что ответ от iiko.biz пришел с кодом «httpStatusCode»: 200. Только в этом случае можно использовать полученный токен.

Если же при получении токена ответ вернулся с каким-то другим кодом, то нужно запросить маркер доступа ещё раз.
