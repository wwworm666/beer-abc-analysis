В этом разделе приводится список базовых вопросов, которые возникают в начале работы с API. На каждый вопрос дан краткий ответ, а также приведены ссылки на разделы руководства, в которых можно найти подробную информацию о решении соответствующей задачи.

**В:** В чём разница между представленными API (iikoServer API, iikoFront API и iikoCloud API)?
 
О: **iikoServer API** предназначен для работы с различными отчетами, которые можно сформировать в iikoOffice. Это могут быть отчеты по сотрудникам, по их сменам, по выручке, получение списков поставщиков, работа с документами (накладные, данные по инвентаризации), работа с номенклатурой и т.п. Также здесь доступно выгрузка OLAP-отчетов.

**iikoFront API** необходим для разработки плагинов для расширения возможностей кассы.

**iikoCloud API** применяется для работы с внешними заказами. При помощи данного API происходит работа с заказами на стол, на доставку, на самовывоз; имеется возможность работы с основными функциями лояльности (расчеты бонусов, получение программ и т.д.).

**В:** Я пишу интеграцию и хотел бы получить доступ к системе. Как и на какой период я могу получить доступ? Сколько это стоит и что в него входит?
 
О: В целях разработки, тестирования мы можем предоставить **бесплатный** демо-стенд с iikoOffice, iikoFront, а также будет доступен для работы iikoWeb (доступен функционал тарифа [Enterprice](https://iiko.ru/solutions/products/price)). Для этого необходимо **[заполнить анкету из статьи](/articles/api-documentations/kak-poluchit-demo-stend)** и прислать нам на e-mail [api@iiko.ru](api@iiko.ru)

Стенд создается под конкретный запрос и будет доступен до тех пор, пока на нем есть активность. Если он станет недоступным (ушел в архив) - сообщите нам, мы продлим лицензию. Также, при необходимости, мы можем обновить версию выданного нами демо-стенда.

На демо-стенде будут предустановлены **все необходимые лицензии** для работы с системой, а также возможна работа в режиме отладки.

**В:** Хочу получить лицензию разработчика

О: Доступные для распространения лицензии приведены на странице **[Какими бывают лицензии API](/articles/api-documentations/~licenzii-api)**.

Если вас интересует персонализированная лицензия от iiko для вашего плагина, напишите запрос на e-mail [api@iiko.ru](api@iiko.ru)с рассказом о продукте (можно прислать презентацию или сайт), вас свяжут с менеджером по лицензированию.

**В:** Можно ли использовать только iikoWeb при работе с демо-стендом?

О: Нет, установка приложений iikoOffice и iikoFront обязательна (см. [инструкцию](/articles/api-documentations/kak-ustanovit-demo-stend)).

Предоставленный демо-стенд содержит функционал тарифа [Enterprice](https://iiko.ru/solutions/products/price), который подразумевает использование [iikoOffice](/articles/iikooffice-9-x/getting-started)как управленческий механизм (ведение номенклатуры, установка цен, управление доступами, сотрудниками, построение отчетов и т.д.), а [iikoFront](/articles/iikofront-9-x/getting-started)представляет собой кассу, на которая работает с заказами.

В контексте работы с демо-стендом iikoWeb выступает как механизм для работы с внешними заказами по [API](/articles/api-documentations/obschee-opisanie-i-vozmozhnosti).
