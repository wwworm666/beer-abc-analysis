* [Установка и настройка Postman](/articles/api-documentations/ustanovka-i-nastroyka-postman/a/h2__403211230)

**Postman** - это платформа для создания и использования API. Postman упрощает каждый этап жизненного цикла API и оптимизирует совместную работу.

Удобное хранение, каталогизация и совместная работа со всеми артефактами API на одной центральной платформе. Postman позволяет хранить и управлять спецификациями API, документацией, рецептами рабочих процессов, тестовыми случаями и результатами, метриками и всем остальным, что связано с API.
