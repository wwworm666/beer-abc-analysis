В данной статье описан процесс настройки оповещения в iikoWeb для оперативной работы по поступающим событиям. Подробнее об событиях можно прочитать в данной [статье](/articles/iikoweb/notification).

Первое, что необходимо сделать перед тем, как использовать доставку, закупки, инвентаризации, прогнозирование, КС и проверку расхождений – **это настроить оповещения по используемому функционалу**.

В iikoWeb в блоке "Настройка и обслуживание системы" выберите "Уведомления" – Подписки на оповещения.

Пример настройки оповещений для ошибок создания доставочных заказов на сайте e-comm – на скрине.

*Название подписки* - название подписки.

*Канал* – интеграция с сервисами доставки, Внешние системы, iikoTransport и тп

*Приоритет* – высокий только для ошибок, низкий – для всех оповещений о создании доставочного заказа

*Чаты* – добавьте свой созданный канал (уведомления – настройки telegram)
