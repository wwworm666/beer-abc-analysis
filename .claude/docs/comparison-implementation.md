### 2. Новая функциональность

**Автоматический выбор периодов:**
- При загрузке автоматически выбирается текущая vs предыдущая неделя
- Режимы сравнения автоматически подбирают нужные периоды

**Умные режимы сравнения:**
- Week-over-Week (текущая vs предыдущая неделя)
- Month-over-Month (текущая vs 4 недели назад)
- Year-over-Year (текущая vs 52 недели назад)

**AI-анализ:**
- Автоматическое выявление значимых изменений (>5%)
- Генерация инсайтов по выручке, среднему чеку, структуре
- Контекстные рекомендации

**Автоматическое сравнение:**
- Сравнение запускается автоматически при изменении селекторов
- Не нужно нажимать кнопку "Сравнить" каждый раз

## 🚀 Как запустить

1. Убедитесь, что Flask сервер запущен:
   ```bash
   python app.py
   ```

2. Откройте браузер: `http://localhost:5000/dashboard`

3. Перейдите на вкладку "Сравнение"

4. Периоды должны выбраться автоматически, сравнение запустится
