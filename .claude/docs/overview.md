# Overview — Общая архитектура

## Что это

Система аналитики продаж пива для сети из 4 баров. Flask-приложение, которое берёт данные из iiko и превращает их в понятные отчёты.

## Архитектура

```
┌─────────────────┐     ┌─────────────────┐
│   Браузер       │────▶│  Flask (app.py) │
└─────────────────┘     └────────┬────────┘
                                 │
        ┌────────────────────────┼────────────────────────┐
        │                        │                        │
        ▼                        ▼                        ▼
┌───────────────┐      ┌─────────────────┐      ┌──────────────┐
│   iiko API    │      │  JSON-файлы     │      │ Google Gemini│
│ (продажи)     │      │ (планы, краны)  │      │ (AI-анализ)  │
└───────────────┘      └─────────────────┘      └──────────────┘
```

## Файлы

```
beer-abc-analysis/
├── app.py              ← Главный файл. Все API endpoints.
├── core/               ← Бизнес-логика
│   ├── iiko_api.py     ← Подключение к iiko
│   ├── olap_reports.py ← OLAP-запросы
│   ├── abc_analysis.py ← ABC-анализ
│   ├── xyz_analysis.py ← XYZ-анализ
│   └── ...
├── templates/          ← HTML-страницы
├── data/               ← JSON-данные
└── .claude/            ← Эта документация
```

## Технологии

| Технология | Почему |
|------------|--------|
| **Flask** | Простой, достаточно для наших задач |
| **Pandas** | Стандарт для работы с таблицами |
| **iiko REST API** | Источник данных о продажах |
| **Google Gemini** | AI-анализ трендов (бесплатный tier) |
| **Render** | Хостинг с постоянным диском |
