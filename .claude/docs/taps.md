# Управление кранами

## Что это

Система учёта 60 разливных кранов в 4 барах. Отслеживание подключённых кег, история замен, статистика активности.

## Файлы

| Файл | Описание |
|------|----------|
| `core/taps_manager.py` | TapsManager — логика кранов |
| `templates/taps_main.html` | Выбор бара |
| `templates/taps_bar.html` | Краны бара |
| `data/taps_data.json` | Состояние кранов |
