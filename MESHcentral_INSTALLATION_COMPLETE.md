## Как установить агент на ПК

### ПАПКА С УСТАНОВЩИКОМ

Все файлы для установки агента находятся в папке:
```
MeshCentral-Agent-Install\
```

**Файлы в папке:**
- `install.bat` — автоматическая установка (запустить от администратора)
- `install-agent.ps1` — PowerShell скрипт установки (от администратора)
- `uninstall.bat` — удаление агента
- `check-status.bat` — проверка статуса
- `00_ГЛАВНАЯ.md` — навигация по всей документации
- `БЫСТРЫЙ_СТАРТ.txt` — краткая шпаргалка
- `ШПАРГАЛКА_КОМАНДЫ.txt` — все команды в одном файле
- `ПОЛНАЯ_ИНСТРУКЦИЯ.md` — полная документация системы
- `АВТОМАТИЗАЦИЯ_API.md` — API и автоматизация через командную строку

### Вариант 1: Автоматическая установка (рекомендуется)

1. Скопируйте папку `MeshCentral-Agent-Install` на целевой ПК
2. Нажмите правой кнопкой на **`install.bat`**
3. Выберите **"Запуск от имени администратора"**
4. Дождитесь завершения установки
5. Проверьте устройство в веб-интерфейсе

### Вариант 2: Через веб-интерфейс (рекомендуется)

1. Войдите в веб-интерфейс: https://103.85.115.227
2. Нажмите **"My Devices"** → **"Add Device Group"** (если ещё не создано)
3. В группе устройств нажмите **"Add Devices"**
4. Выберите тип ОС (Windows/Linux/macOS)
5. Скачайте агент и установите на целевой ПК

### Вариант 2: Прямая загрузка агента

**Для Windows x64:**
```
https://103.85.115.227/meshservice?id=1
```

**Для Linux x64:**
```
https://103.85.115.227/meshservice?id=2
```

После скачивания выполните установку:

**Windows:**
```cmd
MeshAgent.exe install
```

**Linux:**
```bash
chmod +x MeshAgent
sudo ./MeshAgent install
```

## Управление сервисом

```bash
# Проверка статуса
systemctl status meshcentral

# Перезапуск
systemctl restart meshcentral

# Остановка
systemctl stop meshcentral

# Просмотр логов
journalctl -u meshcentral -f
```

## Безопасность

1. **Смените пароль администратора** после первого входа
2. **Включите 2FA** в настройках аккаунта
3. **Настройте IP Filtering** при необходимости
4. Регулярно делайте backup базы данных:
   ```bash
   cp /opt/meshcentral/meshcentral-data/meshcentral.db /backup/meshcentral-backup-$(date +%Y%m%d).db
   ```
