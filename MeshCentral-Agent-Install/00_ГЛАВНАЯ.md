## 🚀 Быстрый старт

### 1. Доступ к серверу

**Веб-интерфейс:** https://103.85.115.227

**Логин/Пароль:**
- Логин: `wwworm37@gmail.com` (или `admin`)
- Пароль: ваш пароль

### 2. Установка агента на новый ПК

**Вариант A — Автоматически:**
1. Скопируйте папку `MeshCentral-Agent-Install` на целевой ПК
2. Запустите `install.bat` **от имени администратора**

**Вариант B — Вручную:**
```powershell
# PowerShell от администратора
$url = "https://103.85.115.227/meshservice?id=1"
$output = "$env:TEMP\meshagent.exe"
Invoke-WebRequest -Uri $url -OutFile $output
Start-Process -FilePath $output -ArgumentList "-fullinstall" -Wait
```

### 3. Проверка

1. Откройте https://103.85.115.227
2. Раздел **"My Devices"**
3. Устройство должно появиться через 10-30 секунд

## 🔐 Безопасность

1. **Смените пароль** после первой настройки
2. **Включите 2FA** в настройках аккаунта
3. **Не передавайте** файлы агента посторонним
4. **Регулярно делайте бэкап** базы данных

**Бэкап на сервере:**
```bash
ssh root@103.85.115.227
cp /opt/meshcentral/meshcentral-data/meshcentral.db /backup/meshcentral-$(date +%Y%m%d).db
```
