## Общая информация

**MeshCentral** — это система удалённого управления компьютерами с открытым исходным кодом.

**Возможности:**
- ✅ Удалённый рабочий стол (через браузер)
- ✅ Терминал/командная строка
- ✅ Файловый менеджер
- ✅ Управление питанием (вкл/выкл/перезагрузка)
- ✅ Мониторинг системы (CPU, RAM, диск)
- ✅ Выполнение команд на удалённых ПК
- ✅ Группы устройств
- ✅ Двухфакторная аутентификация

**Преимущества:**
- Работает через интернет (не требует VPN)
- Не требует открытия портов на клиентских ПК
- Кроссплатформенность (Windows, Linux, macOS, Android)
- Открытый исходный код
- Бесплатно

## Установка агента

### На Windows

#### Способ 1: Автоматическая установка

```cmd
:: Запустить от имени администратора
install.bat
```

#### Способ 2: Вручную

```cmd
:: 1. Скачать агент
powershell -Command "Invoke-WebRequest -Uri 'https://103.85.115.227/meshservice?id=1' -OutFile '%TEMP%\meshagent.exe'"

:: 2. Установить (от администратора!)
%TEMP%\meshagent.exe -fullinstall

:: 3. Проверить
%TEMP%\meshagent.exe -state
```

#### Способ 3: Через PowerShell

```powershell
# Запустить от имени администратора
$url = "https://103.85.115.227/meshservice?id=1"
$output = "$env:TEMP\meshagent.exe"
Invoke-WebRequest -Uri $url -OutFile $output
Start-Process -FilePath $output -ArgumentList "-fullinstall" -Wait
Get-Service "Mesh Agent Service"
```

### На Linux

```bash
# Скачать агент
wget https://103.85.115.227/meshservice?id=2 -O meshagent

# Сделать исполняемым
chmod +x meshagent

# Установить
sudo ./meshagent install

# Проверить
sudo systemctl status meshagent
```

### На macOS

```bash
# Скачать агент
curl -o MeshAgent.pkg https://103.85.115.227/meshservice?id=3

# Установить
sudo installer -pkg MeshAgent.pkg -target /

# Проверить
launchctl list | grep Mesh
```

## Командная строка (MeshCmd)

### Установка MeshCmd на локальный ПК

MeshCmd — утилита для управления MeshCentral из командной строки.

**Скачать:**
```
https://103.85.115.227/meshcmd?id=1
```

Или взять из папки агента:
```
C:\Program Files\Mesh Agent\MeshCmd.exe
```

### Базовые команды

#### Подключение к серверу

```cmd
MeshCmd.exe ConnectServer --url wss://103.85.115.227 --loginuser admin --loginpass "MeshAdmin2026!"
```

#### Получить список устройств

```cmd
MeshCmd.exe GetDeviceList
```

#### Выполнить команду на удалённом ПК

```cmd
:: Синтаксис
MeshCmd.exe Dispatch --deviceid [ID_УСТРОЙСТВА] --meshcmd RunCommand --command "[КОМАНДА]"

:: Пример - выполнить ipconfig
MeshCmd.exe Dispatch --deviceid abc123 --meshcmd RunCommand --command "ipconfig /all"
```

#### Отправить файл на удалённый ПК

```cmd
MeshCmd.exe SendFile --deviceid [ID] --source "C:\file.txt" --destination "C:\Temp\file.txt"
```

#### Скачать файл с удалённого ПК

```cmd
MeshCmd.exe GetFile --deviceid [ID] --source "C:\Windows\win.ini" --destination "C:\Temp\win.ini"
```

#### Перезагрузить удалённый ПК

```cmd
MeshCmd.exe Dispatch --deviceid [ID] --meshcmd RunCommand --command "shutdown /r /t 0"
```

#### Выключить удалённый ПК

```cmd
MeshCmd.exe Dispatch --deviceid [ID] --meshcmd RunCommand --command "shutdown /s /t 0"
```

### Получение ID устройства

**Способ 1: Через веб-интерфейс**
1. Откройте устройство
2. ID виден в URL: `.../General#tab-0000&device=DEVICE_ID`

**Способ 2: Через команду**
```cmd
MeshCmd.exe GetDeviceList > devices.txt
notepad devices.txt
```

**Способ 3: На самом устройстве**
```cmd
"C:\Program Files\Mesh Agent\MeshAgent.exe" -nodeid
```

### Примеры использования

#### 1. Быстрое подключение к рабочему столу

```cmd
:: Запустить просмотр рабочего стола
MeshCmd.exe Desktop --deviceid [ID] --viewonly
```

#### 2. Массовое выполнение команды

```cmd
:: Создать файл со списком ID
echo device1 > devices.txt
echo device2 >> devices.txt
echo device3 >> devices.txt

:: Выполнить команду на всех
for /f %i in (devices.txt) do MeshCmd.exe Dispatch --deviceid %i --meshcmd RunCommand --command "ipconfig"
```

#### 3. Сбор информации со всех ПК

```cmd
:: Создать скрипт
echo @echo off > collect.bat
echo ipconfig /all >> collect.bat
echo systeminfo >> collect.bat
echo dir C:\Users >> collect.bat

:: Отправить и выполнить
MeshCmd.exe SendFile --deviceid [ID] --source "collect.bat" --destination "C:\Temp\collect.bat"
MeshCmd.exe Dispatch --deviceid [ID] --meshcmd RunCommand --command "C:\Temp\collect.bat > C:\Temp\info.txt"
MeshCmd.exe GetFile --deviceid [ID] --source "C:\Temp\info.txt" --destination "info_[ID].txt"
```

## Автоматизация

### PowerShell скрипт для проверки всех устройств

```powershell
# Проверка статуса всех устройств
$devices = MeshCmd.exe GetDeviceList | ConvertFrom-Json

foreach ($device in $devices) {
    Write-Host "Device: $($device.name)"
    Write-Host "Status: $($device.status)"
    Write-Host "Last Connect: $($device.lastConnect)"
    Write-Host "---"
}
```

### Автоматическое оповещение

```powershell
# Проверить offline устройства
$offline = MeshCmd.exe GetDeviceList | Where-Object {$_.status -eq "offline"}

if ($offline.Count -gt 0) {
    $message = "Offline devices: " + ($offline.name -join ", ")
    # Отправить уведомление (email, telegram, etc.)
    Write-Warning $message
}
```

### Расписание задач Windows

```cmd
:: Создать задачу на проверку каждый час
schtasks /create /tn "MeshCentral Check" /tr "powershell -File C:\Scripts\meshcheck.ps1" /sc hourly /ru SYSTEM
```

## Устранение проблем

### Агент не подключается

**Проверка подключения:**
```powershell
Test-NetConnection 103.85.115.227 -Port 443
```

**Перезапуск службы:**
```cmd
net stop "Mesh Agent Service"
net start "Mesh Agent Service"
```

**Полная переустановка:**
```cmd
"C:\Program Files\Mesh Agent\MeshAgent.exe" -fulluninstall
timeout /t 5
rd /s /q "C:\Program Files\Mesh Agent"
:: Скачать новый агент и установить заново
```

### Веб-интерфейс не открывается

**Очистка кэша:**
- Ctrl+Shift+Delete → Очистить кэш
- Или режим инкогнито: Ctrl+Shift+N

**Проверка сервера:**
```bash
# SSH на сервер
ssh root@103.85.115.227

# Проверка статуса
systemctl status meshcentral

# Перезапуск
systemctl restart meshcentral

# Логи
journalctl -u meshcentral -f
```

### Медленное подключение

**Причины:**
- Низкая скорость интернета
- Высокая загрузка CPU на удалённом ПК
- Прокси/фаервол

**Решения:**
- В настройках сессии снизить качество видео
- Отключить обои рабочего стола
- Использовать режим "Только просмотр"

### Ошибки аутентификации

**Сброс пароля администратора:**
```bash
# SSH на сервер
ssh root@103.85.115.227

# Перейти в директорию
cd /opt/meshcentral

# Использовать meshctrl.js
node node_modules/meshcentral/meshctrl.js ResetPassword admin "NewPassword123!"
```

## Безопасность

### Рекомендации

1. **Смените пароль администратора** после первой настройки
2. **Включите 2FA** в настройках аккаунта
3. **Настройте резервное копирование** базы данных
4. **Ограничьте доступ по IP** при возможности
5. **Регулярно обновляйте** MeshCentral

### Резервное копирование

```bash
# Бэкап базы данных
cp /opt/meshcentral/meshcentral-data/meshcentral.db /backup/meshcentral-$(date +%Y%m%d).db

# Бэкап конфигурации
cp /opt/meshcentral/meshcentral-data/config.json /backup/meshcentral-config-$(date +%Y%m%d).json

# Бэкап сертификатов
tar -czf /backup/meshcentral-certs-$(date +%Y%m%d).tar.gz /opt/meshcentral/meshcentral-data/*.crt /opt/meshcentral/meshcentral-data/*.key
```

### Восстановление из бэкапа

```bash
# Остановить сервис
systemctl stop meshcentral

# Восстановить базу
cp /backup/meshcentral-YYYYMMDD.db /opt/meshcentral/meshcentral-data/meshcentral.db

# Запустить сервис
systemctl start meshcentral
```
