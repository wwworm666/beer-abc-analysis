## Как использовать

1. Откройте нужный файл Markdown в своем редакторе
2. Используйте оглавление или функцию поиска для навигации
3. Скопируйте нужные примеры кода
4. Адаптируйте под ваши нужды

## Примечания

- Все файлы конвертированы автоматически из PDF
- Форматирование таблиц может немного отличаться от оригинала
- Для лучшей читаемости рекомендуется использовать Markdown viewer
- Если возникают вопросы, обратитесь к оригинальным PDF файлам
