# Разливное пиво

Аналитика проливов: сколько литров каждого пива продали, сколько кег ушло, какая выручка на литр. Всё, что нужно для контроля разливного ассортимента.

---

## Что показывает

Для каждого пива система считает:

| Метрика | Что значит |
|---------|-----------|
| **Литры** | Общий объём проданного пива (порции * объём порции) |
| **Порции** | Количество проданных порций (0.25л, 0.5л, 1.0л) |
| **Кеги 30л** | Примерное количество кег 30л (литры / 30) |
| **Кеги 50л** | Примерное количество кег 50л (литры / 50) |
| **Доля %** | Доля пива от общего объёма разливного |
| **Недели** | Сколько недель пиво было в продаже |
| **Литры/неделя** | Средний расход за неделю |

Если включены финансовые данные, также отображаются:

| Метрика | Что значит |
|---------|-----------|
| **Выручка** | Сколько заработали на этом пиве |
| **Цена за литр** | Выручка / литры |
| **Себестоимость** | Сколько стоили закупки |
| **Наценка %** | Средняя наценка |
| **Маржа** | Прибыль (выручка - себестоимость) |

---

## Как пользоваться

1. Перейди на страницу **Разливное пиво**
2. Выбери **период** и **бар**
3. Нажми **«Анализировать»**

Результат -- таблица со всеми разливными позициями, отсортированная по объёму (от большего к меньшему).

---

## Как система определяет объём

Название позиции в iiko содержит объём порции:

```
"ФестХаус Хеллес (0,5)"  →  пиво: ФестХаус Хеллес, порция: 0.5 л
"Блек Шип (0,25)"        →  пиво: Блек Шип, порция: 0.25 л
"ХБ Октоберфест (1,0)"   →  пиво: ХБ Октоберфест, порция: 1.0 л
```

Система парсит скобки и извлекает объём в литрах. Поддерживает форматы: `(0,5)`, `(0.5)`, `(500мл)`, `(2)`.

Общий объём = количество порций * объём одной порции.

---

## Для чего полезен

- **Контроль расхода** -- сколько кег уходит за неделю/месяц
- **Сравнение баров** -- где какое пиво популярнее
- **Ценообразование** -- цена за литр показывает, на чём зарабатываем
- **Закупки** -- средний расход за неделю помогает планировать заказы
- **Ассортимент** -- позиции с маленькой долей -- кандидаты на замену
