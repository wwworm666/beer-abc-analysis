# Анализ скидок

Инструмент для маркетолога. Показывает ROI каждой скидки, сегментирует гостей по лояльности и даёт детализацию вплоть до конкретного гостя и его заказов.

---

## Как пользоваться

1. Перейди на страницу **Анализ скидок**
2. Выбери **скидку** из выпадающего списка (загружается автоматически)
3. Выбери **период** и **бар** (или «Все бары»)
4. Нажми **«Анализировать»**

Загрузка может занять 5-15 секунд -- система делает тяжёлый запрос к iiko.

---

## Два режима

### Все бары

Когда бар не выбран:

- **ROI-карточки** -- сводная статистика по скидке
- **Таблица по барам** -- сравнение глубины скидки по точкам (строки кликабельные -- нажми на бар для детализации)
- **Сегментация гостей** -- круговая диаграмма + карточки по сегментам

### Конкретный бар

Когда выбран бар:

- **ROI-карточки** -- статистика только по этому бару
- **Таблица гостей** -- каждый гость с его суммой, визитами, средним чеком (клик открывает детализацию по блюдам)
- **Сегментация гостей** -- по этому бару

---

## ROI-метрики

| Метрика | Формула | Что показывает |
|---------|---------|---------------|
| **Выручка без скидки** | Фактическая выручка + сумма скидки | Сколько бы заплатили без скидки |
| **Выручка со скидкой** | Фактическая выручка | Сколько реально заработали |
| **Отдали скидками** | Сумма всех скидок | Стоимость акции |
| **Глубина скидки** | Скидка / (Выручка + Скидка) * 100% | Насколько «щедрая» скидка |
| **Средний чек** | Выручка / количество чеков | Тратит ли гость больше со скидкой |
| **Чеков** | Количество уникальных заказов | Объём использования |
| **Гостей** | Количество уникальных карт | Охват |

---

## Сегментация гостей

Все гости, использовавшие скидку, делятся на три сегмента по количеству визитов:

| Сегмент | Визитов | Смысл |
|---------|---------|-------|
| **Новички** | 1 | Пришёл один раз -- привлечение |
| **Возвращающиеся** | 2-3 | Начинает ходить -- конверсия |
| **Постоянные** | 4+ | Ходит регулярно -- лояльность |

Для каждого сегмента считается:
- Количество гостей и % от общего
- Общая выручка
- Средний чек

Круговая диаграмма наглядно показывает распределение.

### Как читать

- **Много новичков** -- скидка работает на привлечение, но не удерживает
- **Много постоянных** -- скидка удерживает лояльных, но не привлекает новых
- **Баланс** -- скидка работает на всех этапах воронки

---

## Идентификация гостей

Гости идентифицируются по **карте лояльности** (обычно номер телефона). Гости без карты группируются отдельно как «Без карты».

---

## Для чего полезен

- **Оценка акций** -- стоит ли скидка тех денег, что мы на неё тратим
- **Сегментация** -- понять, кого привлекает скидка
- **Контроль** -- выявить подозрительно высокую глубину скидки в конкретном баре
- **Drill-down** -- посмотреть, что именно берут гости со скидкой
