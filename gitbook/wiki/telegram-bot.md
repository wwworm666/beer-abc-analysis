# Telegram-бот

**@kult_taplist_bot** -- бот, который показывает, что сейчас налито на кранах в каждом баре.

---

## Команды

| Команда | Что делает |
|---------|-----------|
| `/start` | Приветствие и список команд |
| `/taplist` | Меню выбора бара (кнопки) |
| `/taplist1` | Большой пр. В.О |
| `/taplist2` | Лиговский |
| `/taplist3` | Кременчугская |
| `/taplist4` | Варшавская |
| `/taplistall` | Все бары сразу |

---

## Что показывает

Для каждого активного крана бот выводит:

- Номер крана
- Название пива
- Пивоварня
- Стиль (IPA, Lager, Stout и т.д.)
- ABV (крепость)
- Ссылка на Untappd (если есть)

Информация берётся из двух источников:
- **Краны** -- текущее состояние из системы управления кранами
- **Маппинг пива** -- база из 408 записей с информацией о каждом пиве (пивоварня, стиль, ABV)

---

## Как бот находит информацию о пиве

Названия в iiko часто отличаются от «человеческих»:

```
iiko: "КЕГ Бланш де Намур, светлое 30 л"
Маппинг: "Бланш де Намур"
```

Бот использует **нечёткий поиск** (fuzzy matching):
1. Убирает «КЕГ», объёмы, суффиксы (светлое, тёмное)
2. Сравнивает с базой маппинга
3. Если совпадение >= 75% -- считается найденным

---

## Как добавить новое пиво в маппинг

Если бот не показывает информацию о пиве (пивоварню, стиль) -- значит его нет в маппинге.

Добавить можно в файле `data/beer_info_mapping.json`:

```json
{
  "Название из iiko": {
    "brewery": "Пивоварня",
    "beer_name": "Название",
    "style": "IPA",
    "abv": "6.5%",
    "untappd_url": "https://untappd.com/b/..."
  }
}
```

После коммита в GitHub бот автоматически подхватит обновление при следующем деплое.
