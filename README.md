# 🍺 Beer ABC/XYZ Analysis System

**Система комплексного анализа продаж пива и управления разливными кранами для сети баров**

> ⚙️ **Проект работает исключительно через GitHub + Render**
> Развёртывание в облаке, автоматические обновления, никаких локальных установок

---

## 📋 Описание проекта

Веб-приложение на Flask для глубокой аналитики продаж пива в сети из 4 баров с интеграцией iiko API.

### 📊 **Основной функционал - Аналитика**

#### **📈 Дашборд "План vs Факт"**

Интерактивный дашборд для сравнения плановых и фактических показателей:

- **Аналитика**: Основные метрики (выручка, чеки, средний чек, доли категорий)
- **Сравнение**: Сравнение между периодами (неделя к неделе, месяц к месяцу, год к году)
- **Плановые показатели**: Управление планами по неделям
- **Экспорт**: Excel и PDF/HTML выгрузка данных

**Ключевые возможности:**
- Выбор заведения или режим "Общая" (все бары)
- Выбор недели для анализа
- Цветовые индикаторы выполнения плана (зелёный = выполнен, красный = не выполнен)
- Детальная разбивка по категориям (розлив, фасовка, кухня)
- Сохранение и редактирование планов

**API интеграция:**
- Фактические данные загружаются из iiko OLAP отчётов
- Планы хранятся в JSON файле `data/plansdashboard.json`
- Автоматическая синхронизация при изменении периода/заведения

**URL:** `/dashboard`

---

#### **ABC-анализ (трёхбуквенная классификация)**

Классификация каждого сорта пива по трём независимым метрикам:

- **1-я буква**: Выручка (Revenue)
  - A: топ 80% выручки
  - B: следующие 15%
  - C: оставшиеся 5%

- **2-я буква**: Процент наценки (Markup %)
  - A: высокая наценка (>30%)
  - B: средняя наценка (15-30%)
  - C: низкая наценка (<15%)

- **3-я буква**: Маржа в рублях (Margin)
  - A: топ 80% маржи
  - B: следующие 15%
  - C: оставшиеся 5%

**Пример**: `AAA` = топ по выручке + высокая наценка + топ по марже
**Пример**: `CBA` = низкая выручка + средняя наценка + средняя маржа

#### **XYZ-анализ (стабильность спроса)**

Оценка вариативности продаж на основе коэффициента вариации (CV):

- **X**: Стабильный спрос (CV < 10%) - предсказуемые продажи
- **Y**: Умеренная вариация (CV 10-25%) - средняя предсказуемость
- **Z**: Нестабильный спрос (CV > 25%) - непредсказуемые продажи

#### **ABC-XYZ матрица**

Комбинированная классификация для принятия решений:

| Класс | Значение | Действие |
|-------|----------|----------|
| AAA-X | Хит со стабильным спросом | Всегда в наличии, приоритет |
| AAA-Z | Хит с нестабильным спросом | Контролировать запасы |
| CCC-X | Низкая рентабельность, стабильный спрос | Рассмотреть замену |
| CCC-Z | Низкая рентабельность, нестабильный спрос | Вывести из ассортимента |

#### **Анализ разливного пива**

- Отдельный ABC/XYZ для кегов (30L и 50L)
- Сортировка по объёму продаж (литры)
- Расчёт оборачиваемости кег
- Анализ прибыльности по барам

#### **Анализ по официантам**

- Рейтинг официантов по выручке
- Средний чек на официанта
- Количество проданных позиций
- ABC-классификация персонала

#### **Анализ по категориям**

- Сравнение фасовки vs разливное
- Доли в выручке и марже
- Динамика по барам
- Тренды продаж

## 🔧 API Endpoints

### Дашборд (новое!):

- `POST /api/analytics` - Получить фактические данные за период
- `GET /api/plan/<venue>/<period>` - Получить план
- `POST /api/plan/<venue>/<period>` - Сохранить план
- `DELETE /api/plan/<venue>/<period>` - Удалить план
- `GET /api/plans/<venue>` - Все планы для заведения
- `GET /api/weeks` - Список недель
- `POST /api/export/excel` - Экспорт в Excel
- `POST /api/export/pdf` - Экспорт в PDF/HTML

### Аналитика:

- `POST /api/analyze` - Запуск ABC/XYZ анализа
- `GET /api/draft/abc` - ABC-анализ разливного
- `GET /api/waiters/stats` - Статистика официантов
- `GET /api/categories/compare` - Сравнение категорий
- `GET /api/weekly-chart/<bar>/<beer>` - График продаж

### Краны:

- `GET /api/taps/bars` - Список всех баров
- `GET /api/taps/<bar_id>` - Краны бара
- `POST /api/taps/<bar_id>/start` - Подключить кегу
- `POST /api/taps/<bar_id>/stop` - Остановить кран
- `POST /api/taps/<bar_id>/replace` - Заменить сорт
- `GET /api/taps/events/all` - История всех событий

### Номенклатура:

- `POST /api/update-nomenclature` - Обновить из iiko
- `GET /api/beers/draft` - Список разливного пива

## 🎯 Примеры использования

### ABC-анализ фасовки

Определение самых прибыльных позиций:

- **AAA** - топ по всем метрикам → Приоритет для закупок и рекламы
- **CCC** - низкая эффективность → Кандидаты на вывод из ассортимента

### XYZ-анализ для планирования

- **X** - стабильный спрос → Можно планировать закупки заранее
- **Z** - нестабильный спрос → Требуется гибкое управление запасами

### Матрица решений

| ABC-XYZ | Стратегия |
|---------|-----------|
| AAA-X | Всегда в наличии, большие запасы |
| AAA-Y | Регулярный мониторинг, средние запасы |
| AAA-Z | Частые закупки малыми партиями |
| CCC-* | Рассмотреть вывод из ассортимента |
