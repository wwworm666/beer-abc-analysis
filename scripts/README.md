# Scripts

Вспомогательные скрипты проекта, организованные по категориям.

## Структура

```
scripts/
├── debug/          # Отладочные скрипты для диагностики проблем
├── check/          # Скрипты проверки данных и функционала
├── analysis/       # Скрипты анализа данных
├── import_export/  # Импорт и экспорт данных
└── maintenance/    # Обслуживание и настройка
```

## Использование

Все скрипты запускаются из корня проекта:

```bash
# Пример запуска debug скрипта
python scripts/debug/debug_bars.py

# Пример запуска check скрипта
python scripts/check/check_dashboard_simple.py
```

## Категории

### `/debug` - Отладка
Скрипты для диагностики проблем с конкретными барами, данными или функциями.

### `/check` - Проверка
Скрипты проверки корректности данных, маппинга, дашборда.

### `/analysis` - Анализ
Скрипты для анализа продуктов, категорий, полей данных.

### `/import_export` - Импорт/Экспорт
Скрипты для импорта планов из Excel, экспорта данных продаж.

### `/maintenance` - Обслуживание
Batch файлы и скрипты для обслуживания системы:
- `backup.bat` - создание бэкапа
- `daily_update_mapping.bat` - ежедневное обновление маппинга
- `convert_pdf_to_md.py` - конвертация PDF документации в Markdown
